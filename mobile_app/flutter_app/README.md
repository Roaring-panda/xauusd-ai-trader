# Flutter Mobile App

Mobile application for the XAUUSD AI Trader built with Flutter.

## Features

- Real-time account balance and P&L tracking
- Live trading signals
- Trade history and analysis
- Push notifications for alerts
- Settings management

## Setup

### Prerequisites

- Flutter SDK (>=3.0.0)
- Android SDK or iOS SDK
- Xcode (for iOS)

### Installation

```bash
flutter pub get
flutter run
```
